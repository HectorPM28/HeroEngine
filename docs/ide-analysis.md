IDE utilitzats: Visual studio community, visual studio code, Eclipse

Edicio de Codi
Visual Studio Community: El IntelliSense que utilitza no nomes et completa el codi pero tambe o fa segons el context del codi ja creat (tot i que a vegades pot posar coses rares). Te un refactoring molt bo que cambia tots els noms d'una clase a la vegada, pots crear metodes apartats de forma rapida i moure els arxius d'una carpeta a l'altre rapidament.
Visual Studio Code: Tot i que tecnicament pitjor, utilitza el C# Dev Kit que t'ajuda a navegar entre clases mes facilment. Pots crear snippets de forma facil per a tenir mini funcions de codi en segons.
Eclipse: Permeteix una millor navegacio entre pantalles i te una gran refactoritzacio de codi on pots cambiar el nom de variables i aquestes es cambien en tot el codi.

Debuger
Visual Studio Community: te un bo debugger ja que aquest, a mes a mes de deixar-te possar breakpoints, et deixar posar aquests amb condicions per a que el codi nomes pari segons X moments.
Visual Studio Code: Permeteix utilitzar el minim funcional d'un debugger. L'unic problema que es podria trobar es la dificultad de llegir col·lecions.
Eclipse: Com a extra al debugger normal dels dos anteriors Ides, eclipse compta amb una funcio anomenada Call Stack que et deixa veure d'on venen els errors i els metodes que el provoquen.

Executables i dependencies
Visual Studio Community: Controla els executables de forma rapida amb menus desplegables i permeteix posar llibreries rapidament.
Visual Studio Code y Eclipse: Els dos utilitzen les dependencies de forma que aquestes siguin iguals als altres treballadors del equip. Aquests dos tenen una configuracio dels executables millor pero aixo fa que sigui mes complicada.

Control de versions
Visual Studio Community: maravilla de control de versions. Et permeteix fer totes les coses per a fer un projecte amb github i te una bona interfaz.
Visual Studio Code: Permeteix utilitzar GitHub de forma agil, amb colors per guiar i una interfaz sençilla.
Eclipse: Ofereix una bona gestio de branques i conflictes.

Rendiment i Cost
Visual Studio Code: El millor dels tres. No utilitza molta Ram i els temps de carga no son llargs.
Visual Studio Community: Utilizta molta RAM ja que te moltes eines per defecte.
Eclipse: esta en un punt mitja pero la seva configuracio es mes confusa que els altres IDEs.ç

Tots tres son multiplataforma i gratuits menys Visual Studio Community que es exclusiu de Windows, cosa que compensa amb una quantitat de Plugins molt bona.
